# Lista Inteligente 2.0

Aplicação web de gerenciamento de tarefas (to-do list) desenvolvida com tecnologias front-end puras — HTML5, CSS3 e JavaScript ES6+ — sem o uso de frameworks ou bibliotecas externas.

O projeto oferece uma interface moderna e responsiva para criação, edição, exclusão e acompanhamento de tarefas, com persistência de dados automática via LocalStorage do navegador.

## 🔗 Descrição para Currículo

> **Lista Inteligente 2.0** — Aplicação web de lista de tarefas desenvolvida com HTML5, CSS3 e JavaScript (ES6+). Implementa operações CRUD completas (criar, ler, editar e excluir tarefas), persistência de dados com LocalStorage, design responsivo para dispositivos móveis e proteção contra ataques XSS. A aplicação conta com animações CSS, feedback visual do estado das tarefas e suporte a atalhos de teclado, demonstrando domínio de manipulação de DOM, eventos JavaScript e organização de código.

---

## 🚀 Funcionalidades

- **Adicionar tarefas** — via botão ou tecla Enter
- **Editar tarefas** — edição inline com prompt nativo
- **Excluir tarefas** — remoção individual ou de todas de uma vez
- **Marcar como concluída** — com indicador visual (riscado e opacidade reduzida)
- **Persistência de dados** — tarefas salvas automaticamente no LocalStorage
- **Contador de tarefas** — exibição do total de itens na lista
- **Estado vazio** — mensagem informativa quando não há tarefas
- **Animações** — efeitos de entrada suaves ao adicionar novos itens
- **Responsivo** — adaptado para uso em dispositivos móveis
